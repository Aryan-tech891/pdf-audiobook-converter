\# PDF to Audiobook Converter (Python)



A Python application that converts text-based PDF documents into audiobooks.  

The tool extracts text from PDFs, converts it to speech using an offline TTS engine, and exports the final result as a single MP3 audiobook.



---



\## Features

\- Extracts text from text-based PDF files

\- Cleans and preprocesses extracted text

\- Converts text to speech using offline TTS (pyttsx3)

\- Automatically splits long documents into manageable audio chunks

\- Exports WAV files and converts them to MP3 using FFmpeg

\- Merges multiple audio parts into a single audiobook file

\- Fully offline (no API keys required)



---



\## Tech Stack

\- \*\*Python 3\*\*

\- \*\*PyMuPDF (fitz)\*\* – PDF text extraction

\- \*\*pyttsx3\*\* – Offline text-to-speech

\- \*\*FFmpeg\*\* – Audio conversion and merging

\- \*\*PowerShell (Windows)\*\* – Automation and scripting
